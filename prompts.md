## Prompts used

The following are prompts I used during development to
design, implement, debug, and deploy this application. These prompts mainly
guided code structure, and Cloudflare-specific
configuration.

- Give me the fastest way to build an AI-powered app on Cloudflare using Workers AI, Durable Objects for memory, and Pages for a chat UI.
- How should I scaffold a Cloudflare Worker using Wrangler and Workers AI
- Explain how to configure wrangler.jsonc for Workers AI bindings
- How do I add Durable Objects for per-session chat memory in a Cloudflare Worker.
- Why am i getting these errors in my Durable Object setup
- Why am I getting a missing entry-point error with wrangler dev?
- How do I deploy Durable Objects on the Cloudflare free plan using sqlite migrations?
- Why does my Pages frontend fail while curl works, must I add CORS?, how do i do that here?
- Create a minimal static chat UI that works with a Cloudflare Worker API
- How do I connect a Cloudflare Pages frontend to a Worker backend?
